# Bohemian Vnyla

Event discovery and ticketing platform built on AWS serverless architecture.

## Overview

Bohemian Vnyla is a complete event management ecosystem designed for organizers and attendees in South Africa. The platform handles everything from event creation to ticket sales to post-event analytics.

## Architecture

**Backend:**
- AWS Lambda (serverless functions)
- DynamoDB (NoSQL database)
- AppSync (GraphQL API)
- Cognito (authentication)
- S3 (media storage)

**Frontend:**
- React + TypeScript admin dashboard
- Next.js discovery application (SSR)
- React Native mobile app (iOS + Android)

**Integrations:**
- AWS Lex (conversational AI)
- Twilio (WhatsApp bot for personalized recommendations)

## Repositories

- **bv-admin-dashboard** — Event organizer CRM
- **bv-lambda-functions** — Backend business logic
- **bv-discovery-app** — Public-facing event discovery site

## Status

Project was active 2019-2022. Selected repositories sanitized and made public to demonstrate architecture patterns.
